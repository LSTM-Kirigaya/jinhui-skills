# 窗口外框（Tauri 与根视口）

对应配置：`apps/desktop/src-tauri/tauri.conf.json` 中 `app.windows[0]`。

## 关键行为

- **系统原生标题栏关闭**：`decorations: false`，由前端自绘顶栏并自行放置关闭/最小化/最大化。
- **透明窗口**：`transparent: true`，内容区可做圆角或模糊等效果（具体视觉由主题实现）。
- **默认尺寸**：宽约 **1200**，高约 **800**；**最小宽高**约 **800×600**，防止布局挤塌。
- **阴影**：`shadow: false`（当前仓库约束；依构建平台表现可能不同，以实际为准）。
- **圆角与外框来源**：圆角与边框由前端最外层容器实现（推荐 `12px` 圆角 + `1px border`），而非依赖系统窗口装饰。

## 对布局的含义

- 整个 UI 从 **(0,0)** 开始就是应用内容，**没有**系统标题栏占位；顶栏高度完全由前端 `header` 控制。
- macOS 需在顶栏左侧为 **交通灯按钮**预留水平空间，避免与图标/文字重叠（详见 `title-bar` 文件夹）。

## 根 HTML

- `index.html` 仅挂载 `#root`；全屏布局由 React 根组件承担。
- 字体与排版资源在 HTML 中声明；**配色不在本分册讨论**。
