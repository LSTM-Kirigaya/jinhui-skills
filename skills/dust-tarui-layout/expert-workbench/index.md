# 主工作区（ExpertMode）布局

对应实现：`apps/desktop/frontend/src/components/ExpertMode.tsx`。

## 根节点

- **占满父级剩余高度**：`flex-1 flex flex-col min-h-0`。
- **内边距**：约 `p-2`；与外层 `App` 的 `main` 的 `p-4` 叠加，复现时注意是否需合并。

## 上部：核心操作区（`shrink-0`）

1. **路径与操作行（卡片容器）**  
   - 单行 **`flex-wrap`**：左为 **路径输入框**（内嵌文件夹图标），`min-width` 约 **260px**；右为 **按钮组**（选择目录、开始扫描、条件出现的保存快照等）。  
   - 外层卡片：大圆角、细边框、轻阴影（样式自定）。

2. **第二行：模式与统计**  
   - **`justify-between`**：  
     - **左**：圆角容器内 **复选框（浅层目录）** + **竖分割线** + **双选项分段控件**（标准模式 / 开发者模式）。  
     - **右**：扫描完成后 **统计信息块**（耗时、文件数、占用等），可包在 **Tooltip** 内；有批量操作时再出现 **全选 / 执行** 等小按钮。

3. **提示条**：扫描失败、未配置 API 等，使用 **淡入动画** 独占一行，位于操作区下方。

## 中部：进行中状态（互斥）

- **扫描中**：垂直居中 **大图标 + 计数文案 + 脉动条**。
- **AI 分析中**：类似居中占位，替换文案与图标意图。
- **空闲且无结果**：垂直居中 **主按钮（选择文件夹）** + 一行说明文案。

## 下部：扫描结果（`result` 存在且非 AI 分析中）

### 标准模式 + 已有分析结果

- **横向分栏**：`flex` + `min-h-0` + `overflow-hidden`  
  - **左栏固定宽度**约 **`w-80`（320px）**，`shrink-0`，**独立纵向滚动**。  
  - **右栏 `flex-1 min-w-0 overflow-auto`**：卡片列表；宽屏可用 **`grid`** 做双列。

### 开发者模式

- **Treemap 与侧栏** 等左右分栏；可能嵌套 **AI 提示大面板**，内部再用 **响应式双列 grid**。

### 子块通用模式

- **卡片**：圆角尺度分档（`rounded-xl`～`rounded-3xl`）、边框、弱阴影。  
- **分段控件**：浅底圆角轨道 + 两枚互斥 **pill** 按钮，多处复用。

## 嵌套：`AIPromptPanel`

- 大圆角容器、`p-6`、纵向 **`gap`**；上部操作区，中部参数，下部 **大屏双列 / 小屏单列**。
