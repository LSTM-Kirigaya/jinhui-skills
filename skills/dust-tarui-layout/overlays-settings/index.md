# 叠层与对话框（设置、快照、任务队列）

对应实现：`App.tsx` 条件渲染；`AISettings.tsx`、`SnapshotDialog.tsx`、`TaskQueueDialog.tsx` 等。

## 全局叠层

- **设置**：状态为真时挂载 **全屏或大尺寸模态**，覆盖在主内容之上，关闭时卸载。
- **快照管理**：由顶栏快照按钮打开；列表与加载逻辑在独立组件内。
- **任务队列**：由顶栏队列图标打开；列表、暂停、取消等在独立组件内。

## 设置页结构要点

- **横向 Tab** 切换多组配置；每个 **TabPanel** 设 **`height: 100%`、`minHeight: 0`、`overflow: auto`**，小窗口仍可滚动。
- **高级项** 常用 **Accordion** 收纳。
- **嵌套子对话框**（保存预设等）：`maxWidth="sm"`、`fullWidth`，纸面 **圆角约 16px**。

## 通用对话框模式

- **标题行**：可选图标 + 标题。  
- **内容**：表单控件，输入框 **圆角约 12px** 一类尺度。  
- **动作行**：取消/确认，按钮 **`textTransform: none`**、圆角约 **10px**。

## 与主布局关系

- 叠层 **不替换** `header + main` 骨架；主视图仍在下层，由 MUI 管理 **z-index** 栈。  
- 典型为 **居中对话框**，顶栏常仍可见（非全屏遮罩时）。
